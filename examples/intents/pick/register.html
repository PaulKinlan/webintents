Service

This is a service you use. It has registered its intent to be able to handle the pick action for "image/*" datatypes.

Imagine one of the services is a Picasa- or Flickr-style image hosting service. We want to be able to select one of images to use in the application (think attach picture to Gmail).

Pick Intent Registration

<intent 
  action="http://webintents.org/pick" 
  type="image/*" 
  href="pick.html" 
  title="Cat Image picker" />
