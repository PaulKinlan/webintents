Use Cases: Intents

A collection of examples demonstrating use-cases for Web Intents

  • Simple Share - Demonstrates the ability to open an app that shares links
  • Shorten URL - Demonstrates the ability to open an app that shares a link, and the service shortens it and returns a response
  • Pick an Image from a cloud service - Demonstrates the ability to integrate with cloud file hosts and pick a file to include in your client application
  • Discover services providers offer - Demonstrates how you can discover the APIs and services other namespaces offer
